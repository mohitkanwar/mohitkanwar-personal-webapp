My Personal Web App

Welcome to my personal web app repository! This application serves as my personal blog, which is accessible at https://mohitkanwar.com.

Getting Started

To run this application locally, follow these steps:

  1. Clone this repository to your local machine.

    Example :
    git clone https://github.com/mohitkanwar/mohitkanwar-personal-webapp.git

  2. Make sure you have Node.js and npm (LTS version) installed. You can download them from https://nodejs.org.

  3. Open the code in your preferred code editor (e.g., Visual Studio Code).

  4. Install project dependencies using npm:

    Example :
    npm install

  5. Perform an audit fix to ensure your project's dependencies are up to date:

    Example :
    npm audit fix

  6. Install the Angular CLI globally:

    Example :
    npm install -g @angular/cli

  7. Start the application locally:

    Example :
    ng serve -o

  8. Build the application:

    Example :
    npm run build

  9. Generate static pages using Scully:

    Example :
    npx scully

  10. Finally, commit your changes and push them to your repository:

    Example :
    git commit -m "Your commit message"
git push

Your local development environment should now be set up, and you can access the web app by navigating to http://localhost:4200 in your web browser.

Feel free to explore, customize, and contribute to the project. If you encounter any issues or have questions, please don't hesitate to get in touch.
